1. Identify Objectives & Users

วัตถุประสงค์หลัก (Core Objective)

1. เป็น "ศูนย์กลางการหาบ้าน (Adoption Hub)" ให้สัตว์จรจัด เข้าถึงข้อมูลเชิงลึกได้ง่าย
2. เปลี่ยนรูปแบบการบริจาคให้เป็น "Interactive Charity" ส่งอาหารให้มูลนิธิผ่านไลฟ์สด

กลุ่มเป้าหมาย (Target Users)

Primary: กลุ่มคนรุ่นใหม่, นักศึกษา, คนวัยทำงานที่มีใจรักสัตว์ คุ้นเคยกับการไถฟีดวิดีโอ
Secondary: อาสาสมัครและมูลนิธิสัตว์ ที่ต้องการพื้นที่โปร่งใสในการกระจายข่าวสาร

เป้าหมายสูงสุด (Ultimate Goal)

ลดจำนวนสัตว์จรจัดอย่างยั่งยืนผ่านกระบวนการรับเลี้ยงที่ไร้รอยต่อ (Seamless Adoption) และสร้างสังคมที่การช่วยเหลือสัตว์เป็นเรื่องสนุกและทำได้ทุกวัน

2. Research and Analysis

In-depth Interview & Analysis

ลงพื้นที่สัมภาษณ์เชิงลึกกลุ่มเป้าหมาย เพื่อค้นหาว่าทำไม "การรับเลี้ยงสัตว์" จึงเป็นเรื่องยากสำหรับคนรุ่นใหม่

Adoption Pain Point

ขั้นตอนการรับเลี้ยงเดิมยุ่งยาก ข้อมูลสัตว์ไม่ครบถ้วน (เช่น นิสัย, สุขภาพ) และต้องเดินทางไกลไปดูตัวจริง ทำให้หลายคนถอดใจ

Charity Pain Point

การบริจาคเงินแบบเดิมผ่านบัญชีธนาคาร "ไม่เห็นภาพผลลัพธ์" ว่าเงินไปถึงสัตว์จริงไหม ทำให้ขาดความผูกพันและแรงจูงใจ

User Behavior

ผู้ใช้มักไถฟีดวิดีโอดูสัตว์น่ารักช่วง "ก่อนนอน" มีความฝันอยากเปิดฟาร์มแมว แต่มีข้อจำกัดด้านพื้นที่อยู่อาศัย (เช่น อยู่หอพัก)

3. Planning and Design

Primary Persona

"ตัวแทน Gen Z"

นักศึกษา ตัวแทนคนรุ่นใหม่ที่ชอบเสพคอนเทนต์ไวรัล ใส่ใจสังคม แต่มีข้อจำกัดด้านเวลา

Design System

Neo-brutalism + Pastel

ใช้สีสันสดใส ขอบมน เพื่อลดความหดหู่ของประเด็นสัตว์จรจัด สร้างบรรยากาศเชิงบวก

Core App Features

Smart Adoption ค้นหาสัตว์ที่ต้องการบ้าน ใกล้ตัวคุณ พร้อมข้อมูลสถิติเชิงลึกเพื่อการตัดสินใจ
Live Spots ไถฟีดวิดีโอสไตล์ TikTok ส่ง ไอคอนปลา (Virtual Gift) เพื่อสมทบทุนอาหารทันที
Direct Contact มีระบบ Live Chat คุยกับอาสาสมัครตอนสตรีม และปุ่มติดต่อขอรับเลี้ยงได้โดยตรง
Gamification Profile บันทึกสถิติและ Level ความภาคภูมิใจ เปลี่ยนการทำบุญเป็นเป้าหมายที่สนุกสนาน

4. Testing & 5. Analysis

Phase 4: Usability Testing
  • ทดสอบ User Flow: ให้กลุ่มตัวอย่างทดลองกดดูรายละเอียดสัตว์เลี้ยง กด Favorite และส่งของขวัญในห้องไลฟ์
  • Feedback ที่ได้รับ: ผู้ใช้ชอบที่เห็นสถิติสัตว์ชัดเจน ช่วยให้ตัดสินใจรับเลี้ยงง่ายขึ้น และชอบแอนิเมชันปลาเด้งในไลฟ์
Phase 5: Analysis & Iterate

การปรับปรุงจาก Pain Points

ปรับแก้ปุ่ม "Adopt Now" ให้มีขนาดใหญ่และสีตัดกันเด่นชัด (Brutalism) เพื่อกระตุ้น Call to Action ให้เกิดการรับเลี้ยงจริง

แนวทางในอนาคต (Future Vision)

พัฒนา "Video Call Screening" คัดกรองผู้รับเลี้ยง และเชื่อมต่อ Smart IoT Feeder ที่จ่ายอาหารจริงเมื่อมีคนกดส่งของขวัญในแอป

Playful UI

สีสันพาสเทลและดีไซน์ขอบมน (Neo-brutalism) ช่วยดึงดูดความสนใจตั้งแต่หน้าแรก

Smart Discovery

ค้นหาสัตว์เลี้ยงที่อยู่ใกล้ตัว พร้อมหมวดหมู่ที่ใช้งานง่าย แอนิเมชันลื่นไหล

Live Spots Feed

ฟีเจอร์หลัก! ดูไลฟ์สดสไตล์ TikTok เพื่อแชทและแจกอาหารให้น้องๆ แบบเรียลไทม์

Saved Friends

รายการโปรดสำหรับติดตามน้องแมวที่คุณสนใจรับเลี้ยงในอนาคต

Gamification Profile

สะสมประสบการณ์การช่วยเหลือ (Angel Tier) สร้างความภูมิใจในการเป็นผู้ให้

Pet Details

แสดงสถิติและปุ่ม Adopt Now ให้โดดเด่น (Call to Action) พร้อมช่องทางติดต่อโดยตรง
